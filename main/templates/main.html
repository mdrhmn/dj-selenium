Django-Selenium App

This is a sample project to test Selenium web-scraping by extracting FCSIT course timetable from MAYA website. Extraction will be saved in maya.txt.


NOTE: Since MAYA website itself is not optimised/slow, the web-scraping process will take approx. 30-60 secs.

Checklist:

  • Make sure all dependencies are installed (pip install -r requirements.txt)
  • Fill in environment variables in .env

Features:

  • Auto-login (make sure to update your USERNAME and PASSWORD in .env file
  • Auto-filter based on current academic year (2020/2021), semester (2), faculty (FCSIT) and campus (UM-KL)
  • Extract all DataTable rows and auto-click for next page
